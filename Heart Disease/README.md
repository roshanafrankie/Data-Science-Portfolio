# Heart Disease Prediction 🫀

This project focuses on performing **Exploratory Data Analysis (EDA)** and building a predictive model to identify the presence of heart disease based on patient clinical parameters. It transitions from basic data visualization to a structured machine learning pipeline.

## 📌 Project Overview
The goal is to analyze a medical dataset to find correlations between symptoms (like chest pain types) and heart disease, ultimately providing a tool for early detection.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** * `Pandas`: Data manipulation and cleaning.
    * `NumPy`: Numerical operations.
    * `Matplotlib` & `Seaborn`: Data visualization (Pie charts, Bar graphs, Heatmaps).
    * `Scikit-Learn`: Machine learning modeling and evaluation metrics.

## 📂 Features & Visualizations
The project includes several key analysis stages:
* **Data Cleaning:** Automatic removal of missing values (gaps) to ensure statistical accuracy.
* **Distribution Analysis:** A pie chart showing the percentage of patients with vs. without heart disease.
* **Symptom Correlation:** A grouped bar chart analyzing Heart Disease frequency across four Chest Pain types (Typical Angina, Atypical Angina, Non-Anginal, Asymptomatic).
* **Model Evaluation:** Implementation of a Confusion Matrix Heatmap and a Classification Report (Accuracy, Precision, Recall, F1-Score).

## 📊 Key Findings (EDA)
* Visualizes the "Heart Disease" distribution to check for dataset balance.
* Identifies which chest pain types are most strongly associated with positive diagnoses through cross-tabulation.

## 📈 Model Performance
The model is evaluated using:
* **Confusion Matrix:** To visualize True Positives vs. False Positives.
* **Classification Report:** Providing a detailed breakdown of the model's reliability in a medical context.
